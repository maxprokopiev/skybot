# Skybot

Bot for skype with custom commands.

# Setup

Create settings file config/config.yml with initial data:

    ---
    :name: Bot

# Commands

- Bot say hello
- Bot say [something]
- Bot google me [something]
- Bot weather in [City]
- Bot bash me
- Bot deploy branch [branch_name] of [project_name] to [env_name]
- Bot help
